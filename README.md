# 🌟 Stella's Pine Script Starter Kit

**AIフレンドリーなTradingViewストラテジー・テンプレート集**

プログラミング知識がなくても、ChatGPTやClaudeなどのAIに指示するだけでカスタマイズできます。

---

- ✅ **コピペで使える** Pine Script v6コード

- ✅ **暗号資産対応** — BTC/SOL等の設定例付き

## 🚀 使い方

### 方法1: 直接TradingViewに貼り付け

1. TradingViewを開く
2. 下方の「Pineエディタ」タブをクリック
3. テンプレートのコードをコピペ
4. 「チャートに追加」をクリック

### 方法2: AIでカスタマイズ

1. `templates/prompts.md` からプロンプトをコピー
2. ChatGPT/Claudeに貼り付け
3. `[パラメータ]` 部分を好みの値に変更
4. 出力されたコードをTradingViewに貼り付け
