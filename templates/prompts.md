# 🤖 AI用プロンプトテンプレート

以下のプロンプトをChatGPT/Claude等のAIに貼り付けて、
`[ ]` 内の値を変更することでカスタムストラテジーを生成できます。

---

## 📈 SuperTrend戦略カスタマイズ

```
Pine Script v6で以下のストラテジーを作成してください：

## 基本設定
- インジケーター: SuperTrend
- ATR期間: [10-20の値を入力、推奨: 10]
- 乗数: [2.0-4.0の値を入力、推奨: 3.0]

## 追加フィルター
- [ ] ボリュームフィルター（平均の1.5倍以上）
- [ ] RSIフィルター（30-70の範囲外ではエントリーしない）
- [ ] なし

## エントリー条件
- ロング: SuperTrendが上向きに転換
- ショート: SuperTrendが下向きに転換

## 損切り・利確
- 損切り: [ATRベース / パーセンテージ / なし]
- 利確: [リスク比1:2 / パーセンテージ / なし]

## バックテスト期間
- 開始日: [2020-01-01]
- 終了日: [2025-12-31]

コードには日本語のコメントを含めてください。
```

## 📊 RSI + EMA戦略カスタマイズ

```
Pine Script v6で以下のストラテジーを作成してください：

## インジケーター設定
- RSI期間: [7-21の値を入力、推奨: 14]
- EMA期間: [20-200の値を入力、推奨: 50]
- 買われすぎレベル: [70-80、推奨: 70]
- 売られすぎレベル: [20-30、推奨: 30]

## エントリー条件
- ロング: RSIが売られすぎ域から脱出 + 価格がEMAより上
- ショート: RSIが買われすぎ域から脱出 + 価格がEMAより下

## フィルター
- [ ] トレンドフィルター（200EMAより上でのみロング）
- [ ] ボリューム確認

## 損切り・利確
- 損切り: 直近高値/安値の [0.5-2.0] 倍ATR
- 利確: リスク比 [1:1.5 / 1:2 / 1:3]

コードには日本語のコメントを含めてください。
```

## 📦 ボリューム急増検出

```
Pine Script v6で以下のインジケーターを作成してください：

## 設定
- 対象期間: [直近20本]
- 急増判定: 平均の [1.5-3.0] 倍以上

## 表示
- 急増検出時にチャート上に [ラベル / 背景 / マーカー] を表示
- 色: [緑（買い）/ 赤（売り）/ 黄色（中立）]

## アラート
- 急増検出時にアラートを発火

コードには日本語のコメントを含めてください。
```

## 🔧 既存ストラテジーの改良

```
以下のPine Scriptコードを改良してください：

\`\`\`pine
[既存のコードをここに貼り付け]
\`\`\`

## 改良内容
- [ ] パフォーマンス最適化
- [ ] パラメーターの外部化
- [ ] アラート機能の追加
- [ ] 視覚的な改善
- [ ] その他: [具体的に記述]

改良したコードには日本語のコメントを含めてください。
改良点を箇条書きで説明してください。
```
